# Meus Projetos React

Este repositório contém meus projetos de **React**, desenvolvidos como estudos e exercícios práticos. Cada pasta corresponde a um projeto diferente, podendo ser usado como referência para aprender ou praticar React.

---

## Projetos Incluídos

- **[inventory-app](./inventory-app)** – Sistema de controle de estoque.
- **[library-games](./library-games)** – Biblioteca de jogos com listagem e filtro.
- **[password-manager](./password-manager)** – Gerenciador de senhas simples.
- **[primeira pagina](./primeira%20pagina)** – Página inicial de estudo do React.
- **[Profile](./Profile)** – Exemplo de componente de perfil de usuário.
- **[react-rout-app](./react-rout-app)** – Aplicação com **React Router** para navegação entre páginas.
- **[stock-manager](./stock-manager)** – Gerenciador de produtos em estoque.
